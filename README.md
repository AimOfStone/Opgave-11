# Opgave-11
~~ Playlist ~~
The code takes an input file, in this case of songs and their artists, and allows for additions and deletion.
The final output will be printed in the terminal and shown in the output file.

1. Open the input file as a read
2. Iterate through the file, as long as it isn't empty
3. Remove the newline from every line
4. Create the linked lists
5. Open the output file as a write
6. Iterate through the linked lists
7. Add the linked lists to the output file 1 by 1
8. When the list has been iterated through, print the tracks
9. end program
